# Bilibili 视频播放器项目

一个现代化的视频播放平台，基于 Bilibili API 开发，提供以下核心功能。

## 核心功能

### 1. 用户登录
- 基于 Bilibili 二维码登录
- 支持登录状态检查

### 2. 视频推荐
- 获取 Bilibili 推荐视频列表
- 支持无限滚动加载
- 自动去重机制

### 3. 视频搜索
- 根据关键词搜索 Bilibili 视频
- 支持分页加载

### 4. 视频播放
- 使用 HTML5 原生 video 标签
- 支持倍速播放、全屏、弹幕等功能

### 5. 视频弹幕
- 获取视频弹幕数据
- 支持分段加载

### 6. 视频互动
- 支持点赞、投币等互动操作
- 获取视频关系状态

### 7. 相关视频
- 获取与当前视频相关的视频列表
